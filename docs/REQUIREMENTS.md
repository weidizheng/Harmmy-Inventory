# 当前范围与后续计划

## 当前生产范围

- 一个启用仓库：`Montery Park`。
- 38 个 Best Seller 商品和私有主图。
- 中文名优先、英文名辅助；支持 SKU、中文、英文和 IP 搜索。
- 箱、端、盒三个物理库存层级。
- 手机/电脑批量调整库存，提交前汇总核对。
- 手机新增商品和图片。
- 商品卡片显示原表 Details、原表 Quantity/Carton 和箱规计算。
- 独立商品编辑页可修改主图、中英文名、IP、尺寸、价格和箱规。
- Henry、Terrence、Harmmy 密码登录；权限相同。
- 全部业务写入记录数据库操作日志。
- Vercel 生产部署和 Supabase 云数据库。

## 近期优先级

1. 完成一次 Montery Park 全仓盘点和抽查。
2. 建立固定的数据库备份/恢复演练。
3. 根据首次实际盘点反馈优化手机端操作速度与卡片信息密度。

## 后续可选功能

- 多仓库选择与仓间调拨；
- 扫码查找 SKU；
- 商品停用/归档和重复 SKU 合并流程；
- 低库存提醒、盘点差异报表和 CSV 导出；
- 更细的角色权限（仅在业务需要时启用）。

未经明确需求，不提前增加复杂角色或自动拆箱规则。
